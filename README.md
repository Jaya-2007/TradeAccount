A simple Java console-based application that accepts trade account details from the user and displays cash or margin information in a formatted output. This project is ideal for beginners learning Java and object-oriented programming.

## Features

* Takes account details from console input
* Supports Cash and Margin account types
* Displays formatted account summary
* Simple and easy-to-understand code
* Beginner-friendly

##Technologies Used

* Java
* Scanner class for input
* Console output formatting

## How to Run

1. Clone the repository:

   bash
   git clone https://github.com/your-username/TradeAccount-Java.git

2. Open the project in any Java IDE (VS Code)
3. Compile and run:

bash
   javac TradeAccount.java
   java TradeAccount
